## ‚ú® Key Features Demonstrated
1. **Vertical Snack-Feed**: Snap-to-screen vertical scrolling for discovery (TikTok style).
2. **Gesture Navigator**:
   - **Swipe Right**: Like a place (Feedback & Animation).
   - **Swipe Left**: Save to Library for later.
   - **Tap CTA**: Instantly add to your current Trip.
3. **Smart Library**: Grouped by city recommendations with "Add All to Trip" functionality.
4. **Dynamic Itinerary Builder**: Day-wise planning with Morning/Afternoon/Evening slots and lock toggles.
5. **Live Trip Mode**: Real-time "Today" view with timeline navigation and walking indicators.

## üíª Running Locally

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Start the development server**:
   ```bash
   npm run dev
   ```
4. **Open in your browser**:
   Navigate to `http://localhost:3000` and toggle Mobile View (Cmd+Shift+M).
